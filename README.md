# ProjetPortailWebEntreprise

Auteur du projet: PASQUALOTTO Tristan

Ce projet est réalisé en angular et l'hébergement se fait avec firebase.

## Prérequis

Afin d'exécuter les commandes qui vont suivres il est néccéssaire d'avoir installer:
- Node.js
- Npm
- Git

## Installation

Voici les étapes à suivre afin de faire tourner le projet en local.

### Recupérer les sources

Copier coller cette ligne de code dans un terminal afin de cloner le répertoire git sur votre machine:

```
git clone https://github.com/tristan57360/ProjetPortailWebEntreprise.git
```

Se rendre dans le dossier cloné puis faire cette commande:

```
npm install
```

Une fois les dépendances installées. Il est temps de démarrer l'application avec la commande suivante:

```
npm start
```

L'accès à l'application se fait depuis un navigateur à l'adresse suivante:

```
http://localhost:4200/
```

Sur cette page il faut s'authentifier pour accèder au dashboard. Depuis le dashboard il est possible d'accéder aux pages suivantes:
- Utilisateurs: Permet de gérer les utilisateurs de l'application ou son propre profil
- Actualités: Permet de consulter les actualités et d'en ajouter une pour les administrateurs
- GED: Permet de consulter la liste des documents de l'application et d'en ajouter pour les employés ou les administrateurs
